## 아이디어 1: Cinematic Dark Studio

**Design Movement**: 시네마틱 다크 + 산업적 모더니즘 (Industrial Cinema Modernism)

**Core Principles**:
- 깊은 흑색 배경에 황금빛 강조 — 영화 스튜디오의 전문성을 시각적으로 표현
- 날카로운 타이포그래피와 정밀한 그리드 — 장비의 정밀함을 반영
- 미묘한 노이즈 텍스처와 그라디언트 — 필름 그레인의 감성

**Color Philosophy**:
- 배경: `#0A0A0A` (거의 순수 검정)
- 카드: `#141414` (약간 밝은 검정)
- 강조: `#F5C518` (IMDb 골드 — 영화 산업의 황금 기준)
- 보조 강조: `#E8A000` (따뜻한 앰버)
- 텍스트: `#FFFFFF` / `#B0B0B0`

**Layout Paradigm**:
- 비대칭 히어로 섹션 (텍스트 좌측 60%, 이미지 우측 40%)
- 상품 그리드는 5열 마소너리 레이아웃
- 섹션 구분에 대각선 클립 패스 사용

**Signature Elements**:
- 황금색 수평선 구분자 (1px)
- 카드 호버 시 황금 테두리 + 상향 이동 애니메이션
- 로고 옆 필름 스트립 모티프

**Interaction Philosophy**:
- 모든 인터랙션은 영화적 전환처럼 부드럽고 의도적
- 장바구니 추가 시 골드 파티클 효과

**Animation**:
- 페이지 진입: 요소들이 아래에서 위로 페이드인 (stagger 0.1s)
- 카드 호버: translateY(-4px) + 골드 박스 섀도우
- 배너 슬라이더: 크로스페이드 전환

**Typography System**:
- 헤딩: `Bebas Neue` (강렬한 압축 디스플레이 폰트)
- 서브헤딩: `Oswald` (중간 무게)
- 본문: `Noto Sans KR` (한국어 가독성 최적화)
- 가격: `Space Mono` (모노스페이스 — 정밀함 강조)

## 아이디어 2: Brutalist Tech Noir

**Design Movement**: 브루탈리즘 + 테크 누아르 (Brutalist Tech Noir)

**Core Principles**:
- 날 것의 솔직함 — 장식 없이 기능 자체가 미학
- 강렬한 타이포그래피 대비 — 크고 굵은 텍스트가 레이아웃을 지배
- 하드 엣지 컨테이너 — 둥근 모서리 없음, 모든 것이 각지고 선명

**Color Philosophy**:
- 배경: `#111111`
- 강조: `#FFFF00` (순수 노란색 — 경고 테이프처럼 강렬)
- 텍스트: `#FFFFFF`
- 보조: `#FF3300` (경고 빨강 — 할인/특가 표시)

**Layout Paradigm**:
- 신문식 다단 레이아웃
- 상품 카드에 두꺼운 테두리 (2px solid)
- 헤더에 마퀴 텍스트 스크롤

**Signature Elements**:
- 굵은 검정/노랑 경계선
- 타이포그래피 기반 히어로 (이미지 없이 텍스트만)
- 스캔라인 오버레이 효과

**Interaction Philosophy**:
- 즉각적이고 거친 반응 — 부드러운 전환 없음
- 클릭 시 강렬한 색상 반전

**Animation**:
- 글리치 효과 (텍스트 위치 미세 이동)
- 호버 시 색상 즉각 반전

**Typography System**:
- 헤딩: `Black Han Sans` (한국어 블랙 웨이트)
- 본문: `IBM Plex Mono` (모노스페이스)

## 아이디어 3: Premium Dark Elegance (선택)

**Design Movement**: 프리미엄 다크 미니멀리즘 (Premium Dark Minimalism)

**Core Principles**:
- 절제된 럭셔리 — 공간과 여백이 프리미엄을 말함
- 정밀한 타이포그래피 계층 — 크기와 무게로 정보 위계 구성
- 섬세한 골드 강조 — 과하지 않게, 포인트로만 사용
- 상품 이미지가 주인공 — UI는 상품을 돋보이게 하는 무대

**Color Philosophy**:
- 배경: `#0D0D0D` (깊은 검정)
- 카드: `#1A1A1A` (미묘하게 구분되는 어두운 회색)
- 강조: `#D4A017` (앤티크 골드 — 고급스러운 황금)
- 호버 강조: `#F0C040` (밝은 골드)
- 텍스트 주: `#F5F5F5`
- 텍스트 보조: `#888888`
- 구분선: `#2A2A2A`

**Layout Paradigm**:
- 전체 너비 히어로 배너 (풀블리드)
- 상품 그리드: 5열 균일 카드 레이아웃
- 섹션 타이틀: 좌측 정렬 + 골드 하단 라인
- 카테고리 탭: 수평 스크롤 가능한 필 버튼

**Signature Elements**:
- 카드 상단 브랜드 로고 영역 (어두운 배경에 흰색 로고)
- 가격 표시: 골드 색상의 메인 가격 + 취소선 정가
- VVIP 배지: 골드 그라디언트 배경

**Interaction Philosophy**:
- 호버 시 카드 미세 상승 + 골드 테두리 출현
- 장바구니 버튼 호버 시 골드 배경으로 전환
- 부드럽고 우아한 전환 (300ms ease)

**Animation**:
- 스크롤 진입 시 요소 페이드인 + 슬라이드업 (IntersectionObserver)
- 배너 슬라이더: 부드러운 크로스페이드 (5초 자동)
- 카드 호버: translateY(-6px) + box-shadow 확장

**Typography System**:
- 헤딩/로고: `Montserrat` (Bold 700, 기하학적 산세리프)
- 서브헤딩: `Montserrat` (SemiBold 600)
- 본문/UI: `Noto Sans KR` (Regular 400)
- 가격: `Montserrat` (Bold, 골드 색상)
- 영문 강조: `Bebas Neue` (섹션 타이틀용)
